# MCRasp

MCRasp is a Minecraft Classic server software written in Python 3.13.2. It aims to provide a convenient and extensible server implementation with support for **Classic Protocol Extensions (CPE)** (partial), multiple worlds, permissions, and asynchronous networking.

## Features

- Multi-world (multi-map) support
- Permission tree system
- Support for multiple CPE extensions
- YAML-based configuration
- Compatible with ClassiCube and Betacraft
- Fully asynchronous architecture

## Running

From the project directory, run:

### Windows

```bash
python Server.py
```

### Linux

```bash
python3 Server.py
```
